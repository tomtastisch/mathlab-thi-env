# angel – Winkelberechnung über Taylor-Reihe

Konsolenanwendung zur Berechnung des Winkels φ zwischen zwei Punkten  
mittels Reihenentwicklung des Arkustangens. Entwickelt im Rahmen des THI-Praktikums.

## Beschreibung
Das Programm berechnet den Winkel zwischen der x-Achse und der Verbindung  
zweier Punkte im kartesischen Koordinatensystem mit Abbruchbedingung  
\(|a_k| < 10^{-n}\).
